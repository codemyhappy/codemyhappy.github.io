# CodeMyHappy 个人主页

欢迎来到我的个人技术空间。我在这里分享编程经验、技术思考和项目实践。

## 项目特点

- 个性化的设计
- 响应式布局
- 现代化技术栈 (VitePress)
- 易于维护和扩展

## 开始使用

1. 安装依赖：
   ```
   pnpm install
   ```
   或者
   ```
   npm install
   ```

2. 启动开发服务器：
   ```
   npm run dev
   ```

3. 访问 http://localhost:5173 查看网站

## 部署

项目可以轻松部署到任何静态文件托管服务，如 GitHub Pages、Netlify 或 Vercel。

## 文件结构

- `docs/index.md` - 主页文件 (VitePress)
- `docs/.vitepress/config.js` - VitePress 配置文件
- `package.json` - 项目依赖和脚本
- `README.md` - 项目说明
- `LICENSE` - 许可证信息
- `.gitignore` - Git忽略规则
