# 全新开始的网站

这是一个完全重新构建的网站项目，清除了所有历史提交记录，只保留了必要的文件结构。

## 项目特点

- 干净的代码结构
- 现代化的设计
- 响应式布局
- 易于维护和扩展

## 开始使用

1. 安装依赖：
   ```
   npm install
   ```

2. 启动开发服务器：
   ```
   npm run dev
   ```

3. 访问 http://localhost:8080 查看网站

## 部署

项目可以轻松部署到任何静态文件托管服务，如 GitHub Pages、Netlify 或 Vercel。

## 文件结构

- `index.html` - 主页文件
- `package.json` - 项目依赖和脚本
- `README.md` - 项目说明
- `LICENSE` - 许可证信息
- `.gitignore` - Git忽略规则
